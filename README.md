# Приложение для обработки изображений

## Описание проекта

Приложение представляет собой графический редактор изображений, разработанный на языке Python.

Программа позволяет:

* загружать изображения форматов `PNG`, `JPG`, `JPEG`;
* подключиться к веб-камере и сделать с нее фотоснимок;
* отображать изображение в окне приложения;
* показать красный, зеленый или синий канал изображения;
* восстанавливать исходное изображение;
* выполнять обрезку изображения по координатам;
* повышать яркость изображения;
* рисовать зелёную линию с заданными координатами и толщиной.

---

## Используемые технологии

* Python 3.14
* tkinter
* Pillow (PIL)
* OpenCV
* NumPy

## Требования

Необходимо наличие:

* Python версии 3.14 или выше;
* доступа к командной строке операционной системы;
* веб-камеры (для использования функции фотографирования).

## Установка и запуск

### Windows

Создание виртуального окружения:

```cmd
python -m venv venv
```

Активация виртуального окружения:

```cmd
venv\Scripts\activate
```

Установка зависимостей:

```cmd
pip install -r requirements.txt
```

Запуск программы:

```cmd
python main.py
```

---

### Linux / macOS

Создание виртуального окружения:

```bash
python3 -m venv venv
```

Активация виртуального окружения:

```bash
source venv/bin/activate
```

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Запуск программы:

```bash
python3 main.py
```

## Использование программы

### Загрузка изображения

1. Нажать кнопку «Загрузить изображение».
2. Выбрать файл формата PNG, JPG или JPEG.
3. Изображение отобразится в окне приложения.

---

### Получение изображения с веб-камеры

1. Нажать кнопку «Сделать снимок с камеры».
2. После подключения камеры будет получен фотоснимок.
3. Полученное изображение отобразится в окне приложения.

Возможные причины ошибки подключения:

* камера отключена;
* камера используется другим приложением;
* отсутствует разрешение на использование камеры;

---

### Отображение цветовых каналов

Пользователь может выбрать:

* Красный канал;
* Зелёный канал;
* Синий канал.

После нажатия соответствующей кнопки отображается выбранный канал изображения.

---

### Восстановление исходного изображения

Кнопка «Исходное изображение» позволяет вернуть изображение к первоначальному состоянию независимо от выполненных операций обработки.

---

### Обрезка изображения

1. Нажать кнопку «Обрезать изображение».
2. Ввести координаты:

* x1 — левая граница;
* y1 — верхняя граница;
* x2 — правая граница;
* y2 — нижняя граница.

После ввода корректных координат изображение будет обрезано.

---

### Повышение яркости

1. Нажать кнопку «Повысить яркость».
2. Ввести значение увеличения яркости.

Интенсивность каждого цветового канала увеличивается на введённое значение.

---

### Рисование линии

1. Нажать кнопку «Нарисовать зеленую линию».
2. Ввести координаты начала и конца линии.
3. Ввести толщину линии.

После этого на изображении будет построена зелёная линия.

## Обработка ошибок

Программа устойчива к некорректному вводу и выводит сообщения при:

* выборе неподдерживаемого файла;
* отсутствии изображения для обработки;
* невозможности подключения к веб-камере;
* вводе некорректных координат;
* вводе некорректной толщины линии;
* отмене действий пользователем.
